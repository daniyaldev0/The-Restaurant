# Philosophy

The design system should create websites that feel:

• Premium
• Timeless
• Elegant
• Editorial
• Minimal
• Luxurious

Every component should appear handcrafted.

Never design for trends.

Design for longevity.

# Color System

Use CSS Variables.

Example:

--primary

--secondary

--accent

--background

--surface

--surface-alt

--text

--text-muted

--border

--shadow

Never hardcode colors throughout the stylesheet.

Only change variables.

---

# Color Philosophy

Backgrounds should remain calm.

Text should have high readability.

Accent colors should only guide attention.

Never overuse accent colors.

Whitespace is part of the design.

# Consistency Rule

Every new component must visually belong to the existing design system.

Never create a component with different spacing, typography, colors, shadows, or radius.

Consistency is more important than originality.
