# v1.0_EXPERIENCE.md

## Objective

Design and implement a premium **Experience** section for our luxury restaurant website.

This section should communicate that dining here is more than eating—it is a carefully curated experience built around atmosphere, hospitality, craftsmanship, and memorable moments.

Follow every attached project document.

Do **not** modify completed sections unless required for integration.

Generate only the required additions to:

* `index.html`
* `style.css`
* `script.js`

---

# Critical Integration Requirement

The website already contains an **Experience** button/link that appears when the website loads.

This implementation **must integrate with that existing navigation.**

Requirements:

* Give this section the correct unique `id` so the existing Experience button scrolls directly to this section.
* If the existing button already links correctly, preserve its behavior.
* If a minor update is required for proper navigation, modify only what is necessary.
* Do **not** redesign the navigation or Hero section.

The Experience button must work immediately after implementation.

# Animations

Reuse the existing animation system.

Reveal:

* Heading
* Paragraph
* Image
* Experience highlights
* CTA

Use the existing IntersectionObserver implementation.

Animate only:

* opacity
* transform

Use staggered timing.

Animations should replay naturally whenever the section re-enters the viewport.

Maintain approximately 60 FPS.

Respect prefers-reduced-motion.

Default easing:

cubic-bezier(0.22,1,0.36,1)
