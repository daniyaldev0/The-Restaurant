# v1.1_LOCATION.md

## Objective

Design and implement a premium **Location** section for our luxury restaurant website.

The section should make visiting the restaurant effortless while reinforcing the brand's premium identity through elegant presentation, thoughtful information hierarchy, and refined interactions.

Follow every attached project document.

Do **not** modify completed sections unless required for integration.

Generate only the necessary additions to:

* index.html
* style.css
* script.js

---

# Creative Direction

The Location section should not feel like a typical contact section.

It should feel like the final invitation before a guest decides to visit.

The experience should communicate confidence, warmth, and sophistication.

Draw inspiration from:

* Aman Resorts
* Apple
* Nobu
* Four Seasons
* Soho House
* Awwwards-winning hospitality websites

The section should feel:

* Elegant
* Calm
* Editorial
* Minimal
* Timeless
* Premium
* Trustworthy

Avoid:

* Generic Google Maps layouts
* Plain contact cards
* Template footers
* Busy information blocks
* Excessive icons
* Corporate styling

Luxury should come from typography, spacing, photography, and composition.

---

# Section Purpose

Help visitors answer:

* Where is the restaurant?
* When is it open?
* How can I contact it?
* How do I get there?
* Why should I visit?

The information should be immediately understandable without feeling cluttered.

# Animations

Reuse the existing animation architecture.

Reveal:

* Heading
* Supporting paragraph
* Information block
* Map
* CTA

Use the existing IntersectionObserver implementation.

Animate only:

* opacity
* transform

Use staggered timing.

Animations should replay naturally whenever the section re-enters the viewport.

Maintain approximately 60 FPS.

Respect prefers-reduced-motion.

Default easing:

cubic-bezier(0.22,1,0.36,1)
