# UI_UX_PRINCIPLES.md

## Purpose

This document defines the visual and interaction principles for the entire website. Every section should feel as though it was designed by the same team, with the same level of craftsmanship and attention to detail.

The website should communicate luxury through restraint rather than decoration.

---

# Overall Experience

The experience should feel:

* Premium
* Elegant
* Editorial
* Calm
* Sophisticated
* Timeless
* Handcrafted
* Confident

Avoid anything that feels:

* Generic
* Corporate
* Template-based
* Overdesigned
* Cartoonish
* Flashy
* Over-animated

Every design decision should reinforce quality.

---

# Visual Hierarchy

Users should always know where to look first.

Typical hierarchy:

Small section label

↓

Large editorial heading

↓

Supporting paragraph

↓

Primary action

↓

Secondary content

Never compete for attention.

Each section should have one clear focal point.

---

# Layout Philosophy

Prioritize whitespace.

Allow content to breathe.

Use generous vertical spacing.

Avoid cramped layouts.

Content should align to a consistent grid.

Maintain comfortable reading widths.

Sections should transition naturally without feeling disconnected.

---

# Typography

Typography should carry much of the visual identity.

Use elegant serif typography for:

* Primary headings
* Editorial statements

Use modern sans-serif typography for:

* Navigation
* Body copy
* Forms
* Buttons
* Metadata

Avoid excessive font weights.

Favor spacing and scale over boldness.

---

# Cards

Cards should feel refined rather than decorative.

Use:

* Soft shadows
* Large border radius
* Subtle borders
* Generous internal padding

Avoid heavy shadows.

Avoid thick borders.

Cards should appear light and premium.

---

# Buttons

Buttons should feel tactile.

Primary buttons should attract attention without overwhelming the interface.

Hover behavior:

* Slight upward movement
* Soft shadow increase
* Smooth transition

Pressed state:

* Small downward movement
* Immediate feedback

Never use exaggerated scaling.

---

# Forms

Forms should feel welcoming and effortless.

Inputs should:

* Be spacious
* Have consistent height
* Include visible labels
* Display subtle focus states
* Use rounded corners

Validation messages should be calm and informative.

Avoid aggressive colors or intrusive alerts.

---

# Images

Images should feel cinematic.

Prefer:

* High-quality photography
* Natural lighting
* Minimal visual clutter

Images should support the content rather than dominate it.

Maintain consistent aspect ratios where practical.

---

# Icons

Icons should be used sparingly.

Prefer simple line icons.

Icons should support comprehension, not decoration.

Avoid mixing icon styles.

---

# Microinteractions

Every interaction should feel intentional.

Examples:

* Button hover
* Card lift
* Input focus
* Image reveal
* Smooth scrolling
* Navigation feedback

Microinteractions should reward interaction without drawing unnecessary attention.

---

# Animation Philosophy

Animation should enhance clarity.

Never animate purely for decoration.

Animate only:

* transform
* opacity

Motion should feel:

* Smooth
* Natural
* Understated

Animations should never delay usability.

Respect prefers-reduced-motion.

---

# Consistency

Every new section should appear to have been designed at the same time as every previous section.

Maintain consistency in:

* Spacing
* Typography
* Border radius
* Shadows
* Hover behavior
* Animation timing
* Button styles
* Form styles
* Card styles

Avoid introducing new visual patterns unless there is a compelling reason.

---

# User Experience

Users should always understand:

* Where they are
* What they can do
* What happens next

Interfaces should feel intuitive without explanation.

Reduce friction wherever possible.

---

# Premium Details

Quality is communicated through subtle details.

Examples include:

* Precise spacing
* Smooth hover states
* Balanced composition
* Clean alignment
* Consistent rhythm
* Elegant transitions
* High-quality imagery
* Thoughtful typography

Luxury comes from refinement, not excess.

---

# Final Principle

Before completing any section, ask:

"Does this feel handcrafted, calm, timeless, and worthy of a premium restaurant?"

If the answer is no, refine the design until it aligns with the project's standards.
